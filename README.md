# बाल हिंदी वाटिका (Baal Hindi Vatika) - Interactive Hindi Learning App

An interactive and engaging web application designed to help children learn Hindi through fun activities, poems, vocabulary games, and quizzes.

## Features

- **कविता पाठक (Poem Reader):** Read and listen to Hindi poems.
- **शब्द खेल (Word Game):** Interactive vocabulary building games.
- **प्रश्नोत्तरी (Quiz):** Test your Hindi knowledge with fun quizzes.
- **Scoreboard:** Earn stars (तारे) and badges (पदक) as you learn.
- **Interactive UI:** Colorful and engaging interface designed specifically for kids.

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend/Server:** Python (for serving files or specific backend tasks)
- **Fonts:** Google Fonts (Outfit, Yatra One)

## Getting Started

1. Ensure you have Python installed.
2. Run the server using the provided Python script:
   ```bash
   python server.py
   ```
3. Open your browser and navigate to the local server address (usually `http://localhost:8000` or as specified in `server.py`).

## File Structure

- `index.html`: The main entry point and UI structure.
- `index.css`: Styling for the application.
- `index.js`: Interactive logic and app behavior.
- `server.py`: Python backend server.
- `test_tts.py`: Script for testing Text-to-Speech functionality.
- `upload.ps1`: Utility script for uploads/deployment.
